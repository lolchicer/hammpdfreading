### Функционал HammPdfReading

Решение `HammPdfReading` разделяется на несколько частей, выполняющих свои задачи:

1. Чтение
2. Запись
3. Пользовательское десктоп-приложение
4. Web-API
5. Юнит-тесты

## 2. Запись

ПО для записи создаётся в проекте `Infor.HammPdfReading.Csv`.

Проект позволяет записывать детали и страницы с рисунками, считываемые с помощью Решения, в формате CSV и в формате PDF соответственно.
Также ведётся попытка записи рисунков отдельно. Рисунки во всех их форматах выводятся в свои папки в соответствии с их номером и моделью.

Также в Решении остался незадействованный проект `Infor.HammPdfReading.Sqlite`.

## 3. Пользовательское десктоп-приложение

Приложение создаётся в проекте `Infor.HammPdfReading.Gui.Form`.

Приложение даёт пользователю папку с PDF-файлами и выгрузить из них данные с помощью `Infor.HammPdfReading`.
Если в папке уже есть CSV-файл с табличными данными, то данные добавляются в него.
